## Setup
- create a database and save the database name in the .env file
- run php artisan migrate
- run php artisan jwt:secret to generate jwt secret key in the .env file
- configure your mail setting in the .env file
- set your algolia in the .env file
    -ALGOLIA_APP_ID=
    -ALGOLIA_SECRET=
